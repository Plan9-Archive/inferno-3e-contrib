Inferno Contribution: cslookup

Cslookup is an example of a program that interacts directly with /net/cs.

Give it an address (or addresses) in the usual net!address!service form, and it spits out any translations of those addresses in the standard form returned by cs (i.e. clone file and address to connect to).

For instance:
% cslookup tcp!pop3.freeserve.net!pop3
/net/tcp/clone 195.92.193.154!110
/net/tcp/clone 195.92.195.154!110
%
